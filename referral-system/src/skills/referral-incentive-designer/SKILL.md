# Referral Incentive Designer

Design incentive packages based on company profile, referrer profile, and industry norms.

## When to Use

- When designing a new referral program from scratch
- To create personalized incentive packages for specific referrers
- To evaluate and improve existing incentive structures
- To check industry-specific compliance and norms

## MCP Tools (Phase 4)

### `referral_incentive_design_package`
Design a complete incentive package for a specific company + referrer combination.
- Economics: reward ceiling (30% of outbound CAC), annual budget, cost savings
- Package: primary reward, secondary reward, ongoing benefits
- Language guidance and edge case handling

### `referral_incentive_design_program`
Design a complete tiered program with different packages per referrer tier.

### `referral_incentive_evaluate`
Evaluate existing incentive structure and recommend improvements.

### `referral_incentive_get_industry_norms`
Get industry-specific incentive norms and compliance considerations.

## Reward Categories

| Category | Examples | Best For |
|----------|---------|----------|
| Recognition | Case study feature, keynote invite | C-suite, status-motivated |
| Reciprocal | Product credits, premium support | Value-motivated |
| Economic | Gift cards, charitable donation | Manager-level, economic |
| Access | Advisory board, exec network | VP+, relationship-motivated |
| Co-Marketing | Joint webinar, shared content | Marketing-savvy champions |

## Key Economics

- Reward ceiling = 30% of outbound CAC
- Total referral cost should be 70-85% lower than outbound CAC
- Escalation ladder: rewards increase with referral count
